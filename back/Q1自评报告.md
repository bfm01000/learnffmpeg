# Q1 自评报告

## 一、员工工作回顾

Q1 我围绕季度 OKR 推进 KMP SDK 核心能力交付、稳定性提升和用户支持优化，整体目标均已完成。

在交付方面，我完成了尉来版本相关能力推进，支持 Demo 能力建设和第三方 Android Demo App 打包链路搭建，提升了外部演示与接入效率；同时推进 Go Ultra、X5 等机型在拍摄控制、参数配置、模式切换、状态监听和预览能力上的适配与问题收敛，支撑了核心版本交付。

在预览与底层链路能力建设方面，我完成了 Go Ultra 实时预览关键链路打通，包括 H.265 码流获取、解码到 YUV 并完成渲染，解决了丢帧导致的马赛克问题，最终将预览能力推进到稳定可用状态。围绕 X5 全景预览流及车机场景性能问题，我也持续进行了链路分析、性能排查和方案设计。

在稳定性和用户问题治理方面，我处理并推动解决了多类复杂问题，包括预览花屏/黑屏、连接失败、Wi-Fi Crash、ANR、拍照超时、日志过多、编译失败、参数设置异常、Crash 等问题；同时参与了小鹏、比亚迪、长城、海外团队等项目支持，能够结合日志、状态、时序和性能数据推进问题闭环。

本季度我做得比较好的地方主要有三点：

1. 结果导向较强，能优先聚焦交付和客户体验相关的关键问题。
2. 问题定位能力有明显提升，逐步形成了更系统的链路分析思路。
3. 跨团队协同更主动，能够推动设备、固件、SDK、业务侧多方对齐和收敛问题。

需要继续提升的地方主要有两点：

1. 复杂问题的文档化和方法论沉淀还可以更及时、更体系化。
2. 对音视频底层体系的理解还需要继续加深，进一步提升解决复杂问题的上限。

1. **系统强化音视频底层能力。** 结合当前工作和学习计划，继续补强编解码链路、时间戳、音视频同步、像素格式、硬解/软解机制等知识，并通过项目实践把知识转化为能力。

3. **提升复杂项目中的优先级管理与协同推动能力。** 在多项目并行场景下，更好地区分交付重点与长期建设事项，提升任务推进节奏和跨团队协作效率。

长期来看，我希望继续沿着音视频 SDK / 媒体处理 / 底层能力建设方向深入，逐步成长为能够主导复杂链路分析、方案设计和能力沉淀的角色。
